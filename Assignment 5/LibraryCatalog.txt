**Library Catalog System Assignment**

**Objective:**
Develop a library catalog system in Java that allows users to manage books, check out and return books, and search for books based on different criteria. The system should provide a user-friendly interface and maintain accurate records of library resources.

**Features to Implement:**

1. **Book Class:**
   - Create a `Book` class with attributes such as book ID, title, author, genre, and availability status. Implement appropriate getter and setter methods.

2. **Library Catalog:**
   - Implement a class (e.g., `LibraryCatalog`) that manages the library's collection of books. Include methods for:
     - Adding a new book to the catalog.
     - Removing a book from the catalog.
     - Displaying a list of all books in the catalog.

3. **User Interface:**
   - Develop a simple user interface to interact with the library catalog. Use the console for input and output. Allow the user to choose different operations.

4. **Check Out and Return:**
   - Implement methods to handle checking out and returning books. Update the availability status of books accordingly.

5. **Search Functionality:**
   - Provide users with the ability to search for books based on different criteria (e.g., title, author, genre). Display search results clearly.

6. **Data Validation:**
   - Implement validation checks for user input. Ensure that users cannot check out or return books that are not in the catalog.

**Submission Guidelines:**

1. **Code Submission:**
   - Submit the complete Java source code for your library catalog system.

2. **Documentation:**
   - Provide clear documentation explaining the design choices, structure of classes, and any external libraries or APIs utilized (if any).

3. **User Guide:**
   - Include a user guide on how to use the library catalog system. Explain the available operations, input formats, and any error handling.

4. **Testing:**
   - Test your program thoroughly, especially considering edge cases like checking out books that are not available and searching for books with specific criteria.

5. **Reflection:**
   - Write a reflection on the challenges faced, lessons learned, and potential improvements for future iterations of the library catalog system.

**Tips:**
- Break down the problem into smaller tasks before starting the implementation.
- Make use of Java's Object-Oriented Programming principles to design classes and manage book records efficiently.
- Consider implementing data structures like ArrayList or HashMap to store and manipulate book records.

This library catalog system assignment will not only enhance your Java programming skills but also provide practical experience in designing and implementing systems for managing resources. Enjoy working on the assignment!
